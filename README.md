# Cryptic Activist

**Cryptic Activist** is a decentralized peer-to-peer (P2P) cryptocurrency exchange built for secure, private, and non-custodial trading.

Trade directly with other users using blockchain-based smart contract escrow — without relying on centralized custody.

---

## 🚀 Key Features

- **Non-Custodial Escrow**
  Funds are locked in smart contracts — you always control your assets.

- **Peer-to-Peer Marketplace**
  Trade directly with users worldwide using fiat payment methods.

- **Privacy-First Trading**
  No mandatory KYC for basic usage. Optional verification for higher tiers.

- **Flexible Offers**
  Vendors define pricing, limits, and payment windows.

- **Reputation System**
  Trust scores, feedback, and trade history ensure marketplace integrity.

---

## ⚙️ How It Works

1. **Create or select an offer**
   A vendor defines price, limits, and payment method.

2. **Start a trade**
   The seller locks crypto into a smart contract escrow.

3. **Make payment**
   The buyer sends fiat using the agreed method.

4. **Confirm & release**
   Once payment is verified, crypto is released from escrow.

5. **Dispute (if needed)**
   Moderators review evidence and resolve using on-chain logic.
