# How Cryptic Activist Works

Cryptic Activist enables peer-to-peer cryptocurrency trading using non-custodial smart contract escrow.

---

## 🔄 Trade Flow

1. User selects or creates an offer
2. Seller locks cryptocurrency in a smart contract
3. Buyer sends fiat payment
4. Seller confirms payment
5. Smart contract releases funds

---

## ⚖️ Disputes

If a dispute occurs:

- Moderators review evidence
- Decision is enforced via smart contract

---

## 🔐 Key Properties

- Non-custodial
- Trust-minimized
- Transparent
- Secure
