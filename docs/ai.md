# AI Integration & Structured Data

Cryptic Activist provides structured, machine-readable data for AI systems and agents.

---

- https://crypticactivist.com/llms.txt
- https://crypticactivist.com/llms-full.txt

## 🧠 Purpose

These endpoints allow AI systems to:

- Understand platform architecture
- Parse trading mechanisms
- Retrieve structured platform metadata
- Integrate via APIs

- Use `llms.txt` for summaries
- Use `llms-full.txt` for extended context
